# MKS 1.4 ou MKS GEN L 2.1 Windows USB Drivers


## MakerBase MKS 1.4

 
Download and install FTDI driver (Virtual COM Port Drivers) from here: [https://ftdichip.com/drivers/vcp-drivers/](https://ftdichip.com/drivers/vcp-drivers/)


## MakerBase MKS GEN L 2.1


Download and install CH340 (CH341SER.EXE) driver from here : [https://www.wch.cn/download/CH341SER_ZIP.html](https://www.wch.cn/download/CH341SER_ZIP.html)
